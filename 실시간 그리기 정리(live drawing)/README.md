# 개요(Overview)
- 일단 canvas 그리기 구현 필요 (아래 douBoard 참고)

- 실시간 그리기 구현시 socket io 이용해서 canvas 좌표 전송해서 그리기

- 화면 해상도가 각각 다를시 html2canvas를 이용해서 화면 캡처 및 디비 저장 필요
